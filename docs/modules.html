getMealList

  • getMealList(): Promise<MealList>
    • Defined in cafeteria/index.ts:134

    Returns cached list if no updates found, otherwise fetches and scrapes anew.

    Note that there is absolutely no guarantee for this list to be accurate. The data is parsed from a PDF file, which are notoriously hard to deal with. The accuracy depends on a myriad of factors, including the build of poppler's pdftotext you have on your system's path.

    The accuracy seems to be alright with version 0.71.0 and 0.86.1, and probably most versions in between.

    Returns Promise<MealList>

    The current week's meal list
